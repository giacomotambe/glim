Score voxels and split points into static / dynamic frames.

Takes the WallFilterResult produced by WallFilter::filter() for the current scan. The voxelmap inside wf_result is used directly — no additional voxelization is performed.

Wall voxels (is_wall == true) are unconditionally kept as static. All other voxels are scored against the previous frame's voxelmap and classified as static or dynamic based on the configured weights and threshold.

On the first call (empty history) the voxelmap is stored as the reference frame and source_frame is returned unchanged as static output.
