41// ---------------------------------------------------------------------------
42// WallBBoxRegistry
43//
44// Mantiene un registro persistente di bounding box di pareti.
45// Ad ogni chiamata a update() le OBB del frame corrente vengono confrontate
46// con quelle già salvate tramite overlap IoU (Intersection over Union).
47//
48// - Se IoU >= overlap_threshold → la box esistente viene aggiornata
49// (media pesata tra quella salvata e quella nuova, SLERP per rotazione).
50// - Se IoU < overlap_threshold → la box viene aggiunta come nuova parete.
51//
52// L'overlap è calcolato con la Separating Axis Theorem (SAT) su OBB 3D,
53// che gestisce correttamente box ruotate a differenza delle AABB.
54// ---------------------------------------------------------------------------
