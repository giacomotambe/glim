Detailed Description

Error-State Kalman Filter that estimates the absolute pose T_world_imu by fusing IMU predictions with SLAM updates.

Between SLAM updates the filter integrates IMU measurements to accumulate the incremental motion (position delta + rotation delta + velocity).

When a new SLAM pose arrives, update() computes the SLAM-observed delta, fuses it with the IMU prediction, and returns the filtered absolute pose T_world_imu.

Nominal state (relative to previous frame): Δp (3) – position delta Δv (3) – velocity at current time Δq (4) – rotation delta (quaternion)

Error state (9-dim): δp (3), δv (3), δθ (3)
